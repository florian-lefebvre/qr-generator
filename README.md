# QR Code Generator

Generate unlimited QR Codes with ease for free

## Contributing

This website is made using Astro, TypeScript and Tailwind CSS. Install dependencies using pnpm: 

```bash
pnpm i --frozen-lockfile
```

Start the dev server:

```bash
pnpm dev
```
